## Installation & Run Locally

Clone the project

```bash
  git clone https://github.com/Soheil-Saberi/snappfood_demo.git
```

Go to the project directory

```bash
  cd snappfood_demo
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

Open http://localhost:3000 to view it in the browser.
